#shell

一些简单的命令、alias

##alias

    => git clone https://github.com/myhirra/shell
    => sh alias.sh
    => source ~/.zshrc 或者source ~/.bashrc

比如，mac下想直接google的话

    => goole test

##link

    => sh link.sh

比如，terminal中，想直接用sublime打开directory或file

    => sublime codes

##sbn

    sbn add/del #批量执行svn add/del。
    sbn cp ${path} #copy当前目录下所有改动到指定的目录。
